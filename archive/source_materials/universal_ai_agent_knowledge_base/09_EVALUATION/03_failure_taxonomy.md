# Failure taxonomy

## Типы ошибок агента

1. Неверно понял задачу.
2. Пропустил важный файл.
3. Изменил лишние файлы.
4. Сломал public API.
5. Не добавил тест.
6. Не запустил проверку.
7. Скрытая security regression.
8. UX regression.
9. Галлюцинировал результат команды.
10. Написал `готово` без доказательств.

## Использование

Каждый failed run классифицируется по этой таксономии.
