# Agent evaluation plan

## Что оценивать

- корректность планов;
- соблюдение file scope;
- качество diff;
- тесты;
- security awareness;
- UX awareness;
- качество отчётов;
- способность находить свои ошибки;
- количество доработок после агента.

## Метрики

- task success rate;
- regression rate;
- tests added per bugfix;
- forbidden path violations;
- false done count;
- average diff size;
- CI pass rate;
- reviewer rejection rate.
