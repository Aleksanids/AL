# Manifest

Архив содержит базу знаний и шаблоны для создания универсального AI coding agent.

## Принцип разделения

- `knowledge` — методология и стандарты.
- `templates` — файлы, которые Codex может использовать как заготовки.
- `checklists` — приёмка и ревью.
- `sources` — карта источников.

## Что НЕ включено

- секреты;
- токены;
- приватные пути конкретного пользователя;
- привязка к конкретному приложению;
- копии чужих статей;
- длинные цитаты из источников.
