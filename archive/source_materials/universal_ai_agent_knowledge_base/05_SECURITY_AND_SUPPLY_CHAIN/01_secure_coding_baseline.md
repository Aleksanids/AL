# Secure coding baseline

## Проверять всегда

- input validation;
- output encoding;
- authentication;
- authorization;
- session handling;
- cryptography;
- error handling;
- logging;
- data protection;
- file handling;
- database access;
- network calls;
- secrets handling.

## Agent rule

Если код работает с внешними данными, файловой системой, сетью, БД или auth — требуется security review.
