# 05 Testing Checklist

- [ ] Есть тест на новый функционал.
- [ ] Есть regression test для bugfix.
- [ ] Unit tests быстрые.
- [ ] Integration tests изолированы.
- [ ] E2E только для критичных сценариев.
- [ ] Тесты не зависят от реальных секретов.
- [ ] Edge cases проверены.
- [ ] Coverage не падает без объяснения.
