# 07 Ux Ui Checklist

- [ ] Главное действие понятно.
- [ ] Статус системы виден.
- [ ] Есть empty/loading/error states.
- [ ] Ошибки объясняют восстановление.
- [ ] Keyboard navigation работает.
- [ ] Focus visible.
- [ ] Контраст достаточный.
- [ ] Тексты понятны пользователю.
- [ ] Responsive behavior проверен.
- [ ] Нет color-only meaning.
