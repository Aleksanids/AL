# 06 Security Checklist

- [ ] Внешние данные валидируются.
- [ ] Нет secrets в коде/логах.
- [ ] Нет опасных shell-команд.
- [ ] Auth/authz не ослаблены.
- [ ] SQL/command injection проверены.
- [ ] File paths нормализованы.
- [ ] Dependencies проверены.
- [ ] GitHub Actions permissions минимальны.
- [ ] Untrusted issue/PR text не исполняется.
