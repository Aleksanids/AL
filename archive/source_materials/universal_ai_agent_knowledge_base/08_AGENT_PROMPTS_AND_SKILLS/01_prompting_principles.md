# Prompting principles для coding agents

## Хорошая задача содержит

- цель;
- контекст;
- входные данные;
- ограничения;
- запреты;
- порядок действий;
- критерии приёмки;
- команды проверки;
- формат отчёта.

## Плохая задача

```text
Исправь всё и улучши код.
```

## Хорошая задача

```text
Исправь ошибку X. Менять можно только src/module.py и tests/test_module.py. Добавь regression test. Запусти ruff, mypy, pytest. Отчёт: причина, diff, проверки, риски.
```
