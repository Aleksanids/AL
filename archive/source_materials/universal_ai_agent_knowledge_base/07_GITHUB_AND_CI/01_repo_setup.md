# GitHub repo setup

## Обязательные файлы

- README.md;
- LICENSE, если open-source;
- SECURITY.md;
- CONTRIBUTING.md;
- CODEOWNERS, если команда;
- .gitignore;
- pyproject.toml;
- .github/workflows/ci.yml;
- .github/pull_request_template.md;
- .github/ISSUE_TEMPLATE/bug_report.yml;
- .github/dependabot.yml.

## Private by default

Если репозиторий содержит внутренние правила, пути или workflow — лучше private.
