# Управление зависимостями

## Принципы

- фиксировать зависимости lockfile;
- отделять runtime/dev/test dependencies;
- не добавлять зависимость ради одной простой функции;
- проверять лицензию и поддержку;
- использовать Dependabot/security alerts;
- не обновлять всё одновременно с feature/hotfix.

## uv

`uv` можно использовать как быстрый менеджер Python-проектов, зависимостей, lockfile и инструментов.
