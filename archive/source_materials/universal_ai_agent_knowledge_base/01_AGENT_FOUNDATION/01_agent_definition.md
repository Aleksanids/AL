# Что такое универсальный AI coding agent

AI coding agent — это не «чат с моделью», а управляемая система разработки.

## Компоненты

1. **Task intake** — принимает задачу в структурированном виде.
2. **Classifier** — определяет тип задачи.
3. **Context builder** — собирает релевантные файлы, историю, инструкции, ошибки.
4. **Planner** — строит план и scope.
5. **Executor adapter** — передает задачу Codex/Cursor/Aider/GitHub agent.
6. **Diff inspector** — анализирует измененные файлы.
7. **Quality gates** — запускает проверки.
8. **Critic** — ищет ошибки, регрессии и ложную готовность.
9. **Verifier** — принимает работу только по фактам.
10. **Reporter** — формирует отчёт.

## Главный критерий силы агента

Агент силен не тогда, когда генерирует много кода, а когда снижает количество ошибок после генерации.
