# AGENTS.md

## Mission

You are working inside a software repository. Your goal is to improve code quality without breaking existing behavior.

## Mandatory workflow

1. Read project instructions.
2. Understand the task.
3. Identify affected files.
4. Prepare a plan.
5. Respect allowed/forbidden paths.
6. Make minimal changes.
7. Add or update tests.
8. Run quality gates.
9. Review diff.
10. Report facts, not assumptions.

## Never do

- Do not claim success without running checks or explaining why checks could not run.
- Do not change unrelated files.
- Do not remove tests to make CI pass.
- Do not expose secrets.
- Do not run dangerous commands without approval.
