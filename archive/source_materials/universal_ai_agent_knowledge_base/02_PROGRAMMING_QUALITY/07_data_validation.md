# Валидация данных

## Принцип

Все внешние данные недоверенные.

## Проверять

- тип;
- формат;
- диапазон;
- обязательность;
- длину;
- enum;
- кодировку;
- путь файла;
- размер файла;
- источник.

## Agent rule

Если функция принимает данные извне, агент должен добавить или проверить слой валидации.
