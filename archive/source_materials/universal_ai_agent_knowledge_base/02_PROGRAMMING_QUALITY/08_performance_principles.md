# Производительность

## Не оптимизировать вслепую

Сначала измерение, потом оптимизация.

## Проверять

- асимптотика;
- повторные I/O операции;
- лишние сетевые запросы;
- N+1 queries;
- память на больших файлах;
- кэширование;
- lazy loading;
- batch operations.

## Для AI-агента

Если задача работает с большими файлами или списками, агент должен явно проверить сценарий большого объёма.
