# Microcopy

## Правила

- кнопка говорит действие: `Скачать`, `Сохранить`, `Проверить`;
- ошибка объясняет действие: `Проверьте формат файла`;
- не использовать внутренний жаргон;
- не писать «ошибка 500» без понятного объяснения;
- подтверждения нужны для разрушительных действий.

## Agent rule

Если UX затрагивает текст интерфейса, reviewer должен проверить tone, clarity и actionability.
