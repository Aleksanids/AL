# UX baseline для AI coding agent

## Агент должен проверять

1. Пользователь понимает, где он находится.
2. Пользователь видит статус операции.
3. Основное действие очевидно.
4. Ошибка объясняет, что случилось и что делать.
5. Есть пустые состояния.
6. Есть loading states.
7. Нет скрытых критичных действий.
8. Тексты написаны языком пользователя.
9. Интерфейс не требует помнить лишнее.
10. Визуальная иерархия помогает, а не мешает.
