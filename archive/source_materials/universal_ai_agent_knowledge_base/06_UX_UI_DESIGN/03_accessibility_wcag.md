# Accessibility / WCAG baseline

## Проверять

- keyboard navigation;
- focus visible;
- contrast;
- labels;
- alt text;
- semantic markup;
- error messages;
- form instructions;
- resize/reflow;
- screen reader names;
- no color-only meaning.

## Уровень

Для большинства коммерческих интерфейсов базовая цель — WCAG 2.2 AA, если нет других требований.
