# Формы, таблицы, карточки

## Формы

- понятные labels;
- inline validation;
- сохранение введённых данных при ошибке;
- disable submit только с объяснением;
- понятные required fields.

## Таблицы

- видимые заголовки;
- сортировка/фильтры;
- пустое состояние;
- sticky только если не мешает;
- не обрезать критичные данные без tooltip/details.

## Карточки

- главный идентификатор сверху;
- вторичные данные ниже;
- действия отделены от данных;
- статусы визуально различимы и текстово понятны.
