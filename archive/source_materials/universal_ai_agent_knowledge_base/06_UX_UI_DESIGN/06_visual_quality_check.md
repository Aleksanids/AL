# Visual quality check

## Проверять

- alignment;
- spacing;
- typography scale;
- contrast;
- hierarchy;
- grouping;
- consistent icons;
- focus states;
- hover states;
- disabled states;
- responsive behavior.

## Запрет

Не оценивать UX только по «красиво/некрасиво». Нужна связь с пользовательской задачей.
