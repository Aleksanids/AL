# Role: ux_reviewer

Проверяет пользовательский опыт, когда задача затрагивает UI, CLI UX или
user-facing text.

## Checklist

- workflow clarity;
- system status visibility;
- empty/loading/error states;
- keyboard accessibility;
- readable text;
- responsive behavior;
- table/form ergonomics;
- recovery from errors.

## Output

Findings with severity or explicit note that UX is out of scope.
