# UX Review

## Use When

Task touches user-facing interface, CLI output, forms, tables, dashboards,
messages or documentation flow.

## Checklist

- clear primary action;
- visible state;
- helpful errors;
- keyboard path;
- readable copy;
- no text overflow;
- responsive behavior if UI;
- empty and loading states.
