# Bug Investigation

## Workflow

1. Capture observed behavior and expected behavior.
2. Reproduce with command, test or minimal script.
3. Localize to module/function.
4. Separate symptom from root cause.
5. Patch only after evidence.
6. Add regression test.

## Report

- reproduction;
- root cause;
- changed files;
- checks;
- remaining risk.
